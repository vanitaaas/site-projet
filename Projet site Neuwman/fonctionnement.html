Le Cycle d'Instruction

1

Recherche (Fetch)

L'unité de contrôle :

  • Lit l'adresse dans le compteur ordinal (CO)
  • Accède à la mémoire à cette adresse
  • Copie l'instruction dans le registre d'instruction
  • Incrémente le compteur ordinal
2

Décodage (Decode)

L'unité de contrôle analyse l'instruction :

  • Identifie l'opération à effectuer
  • Détermine les opérandes nécessaires
  • Localise les données en mémoire
3

Exécution (Execute)

L'UAL traite l'instruction :

  • Effectue les calculs demandés
  • Réalise les opérations logiques
  • Utilise les registres temporaires
4

Stockage (Store)

Le résultat est enregistré :

  • Stockage en mémoire centrale
  • Mise à jour des registres
  • Actualisation des drapeaux d'état
